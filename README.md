# Minesweeper — Win32 GDI 扫雷游戏

纯 C 语言编写的 Windows 扫雷游戏，采用 Win32 GDI 图形界面，风格对标 Windows 7 经典扫雷。

## 功能特性

### 基础功能
- 经典扫雷玩法：左键揭开、右键插旗、和弦揭开
- 三级难度：初级 (9×9)、中级 (16×16)、高级 (30×16) + 自定义尺寸
- BFS 洪水填充（非递归，避免栈溢出）
- 首次点击安全保护
- 计时器、剩余雷数显示

### 创新功能
- **无猜测模式**：约束求解器数学保证每局可纯逻辑通关，无需猜测
- **智能提示 (H键)**：AI 分析当前局面，高亮一个安全格子
- **概率热力图 (P键)**：每个未揭开格子显示地雷概率（绿=安全，红=危险）
- **战争迷雾 (F键)**：只能看到已揭开区域周围的格子，增加挑战难度
- **九宫格按压反馈**：左键按住数字格，周围格子凹陷显示；旗帜标对后松开自动揭开
- **游戏回放**：录制每次操作，支持保存/加载回放文件 (.msrep)
- **用户登录**：注册/登录/游客模式，个人成绩独立保存
- **排行榜**：每个难度 Top 10，按用时排名
- **局域网对战**：TCP 双人同板竞速，先清完者获胜
- **音效**：异步蜂鸣器音效（后台线程，不阻塞 UI）

## 编译与运行

### 环境要求
- **编译器**：MinGW GCC (推荐 8.0+)
- **操作系统**：Windows 7 / 10 / 11
- **依赖库**：gdi32, winmm, comdlg32, ws2_32 (Windows 系统自带)

### 一键编译
```bat
build.bat
```

### 手动编译
```bash
gcc -Wall -Wextra -O2 -Iinclude -c src/board.c src/solver.c src/game.c src/replay.c src/stats.c src/sound.c src/user.c -o ...
gcc -Wall -Wextra -O2 -Iinclude -DUNICODE -D_UNICODE -D_WIN32_WINNT=0x0600 -c src/render.c src/net.c src/main.c -o ...
gcc -o minesweeper.exe *.o -lgdi32 -lwinmm -lcomdlg32 -lws2_32 -mwindows -municode
```

### 运行测试
```bash
gcc -O2 -Iinclude -o test_core.exe test/test_core.c src/board.c src/solver.c src/game.c -lgdi32
test_core.exe
```

## 核心算法

### 约束求解器 (solver.c)

迭代应用三条推理规则：

1. **安全规则**：数字格周围已标雷数 = 数字 → 其余邻居安全
2. **标雷规则**：数字格周围未揭开数 = 剩余雷数 → 全部是雷
3. **约束相减**：相邻数字格 A、B，若 A 的未揭开集 ⊂ B 的未揭开集 → 差集中雷数 = B余 - A余

### 无猜测生成

```
循环最多 1000 次:
  1. 随机布雷 (避开首次点击九宫格)
  2. 运行求解器
  3. 若求解器能揭开所有安全格 → 成功
```

### 概率热力图

- 边界格子：回溯枚举所有合法布雷方案，精确计算 P(雷)
- 内部格子：P = 剩余雷数 / 内部格子数
- 大组件 (>20格)：随机采样近似

## 操作说明

| 操作 | 说明 |
|------|------|
| 左键点击 | 揭开格子 |
| 右键点击 | 插旗 / 问号 / 取消 |
| 左键按住数字格 | 周围格子按压效果；旗帜标对后松开自动揭开 |
| 左右双键 / 中键 | 和弦揭开 |
| N | 新游戏 |
| H | 智能提示 |
| P | 概率热力图 |
| F | 战争迷雾 |

## 联机对战

1. 玩家 A：菜单 → 联机 → 创建房间 → 显示 IP
2. 玩家 B：菜单 → 联机 → 加入房间 → 输入 IP
3. 双方在相同雷区竞速，先清完获胜
